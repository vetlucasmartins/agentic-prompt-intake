# Cline rule: intake-refiner

This workspace follows the Agentic Prompt Intake Protocol.

When the user's request is vague, voice-transcribed, rambling, incomplete, or missing a clear deliverable, do not execute immediately.

First:

1. Restate the likely intent.
2. Extract a structured brief.
3. Identify critical gaps.
4. Ask concise clarifying questions.
5. Produce a provisional refined prompt or task brief.

Use `AGENTS.md` and `docs/INTAKE-PROTOCOL.md` as the canonical references.
